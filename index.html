Html Table elments

Tag Name Description
<table> Table The wrapper element for all Html tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data inside the table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Basic Html Elements

Tag Name Description
<div> Division The generic container for flow content.
<header> Header The introductory content, typically a group of introductory or navigational aids.
<footer> Footer The footer for its nearest sectioning content or sectioning root element.
<span> Span The generic inline container for phrasing content.
<h1-h6> Headings They are labeled 1-6, with 1 being the largest sized heading and 6 being the smallest.
<p> Paragraph Represents a paragraph of text.
<img> image post image to site
<a> Anchor Defines a hyperlink, which is used to link from one page to another.

Css Basic Properties

Property Description
color: Sets the color of text
background-color: Sets the background color of an element
font-size: Sets the size of the font
margin: Sets the space outside the border of an element
padding: Sets the space between the content and the border of an element
border: Sets the border around an element
